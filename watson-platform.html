Design Watson Platform Experience

IBM Watson is a technology platform that uses natural language processing and machine learning to reveal insights from large amounts of unstructured data. As a user experience designer at Watson Platform, I'm responsible for designing a cognitive system enabling a new partnership between developers and Watson's Language, Vision, Speech and Data APIs. I'm also a front-end developer, constructing and maintaining Watson platform pattern library.

Identify Platform Experience Opportunities

This project is to understand the developer playing field by identifying relevent points of engagement and what role Watson plays in them. The following map provides an overarching perspective of where Watson could have presence and ones that present an opportunity for developer experience.

Design Consistent Watson Experience across Different Cloud Platforms

This year Watson platform's strategy is to scale fast. We will provide Watson APIs in multiple cloud platforms besides Bluemix (IBM Cloud Platform). I'm currently designing the workflow in each cloud platform and responsible for the consistent experience among them.
