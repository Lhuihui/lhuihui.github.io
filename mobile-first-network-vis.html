MVP in an interactive network visualization
What makes an interactive network visualization an interactive network visualization?
It will incorporate at least the following core elements:
  • A way for users to investigate the node
  • A way for users to investigate the relationships between nodes
  • Ways for users to investigate the network - expand, collapse the network
    • Design Assumptions
