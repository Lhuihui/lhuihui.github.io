MVP in an interactive network visualization
What makes an interactive network visualization an interactive network visualization?
It will incorporate at least the following core elements:
  • A way for users to investigate the nodes
  • A way for users to investigate the relationships between nodes
  • Ways for users to investigate the network - expand, collapse the network
    • Design Assumptions
    Avoid hairball
    Users will purposefully explore from relevant portion.
    Don't.
    Impenetrable diagram that is too noisy to be a useful tool for discovery.
    Do.
    Users will purposefully explore from relevant portion.
    Expand in context
    Users will purposefully expand the network while exploring.
    Don't.
    The network visualization tool is a poor substitute for data analytics.
    Do.
    Users will purposefully expand the network while exploring.

    Interaction Guide
    Investigate the nodes
    Left click/tap a node to see information about the node.
    Investigate the relationships between nodes
    Multi-select several nodes to see relationship between them.
    Expand the network
    Double click the node to open the expand menu and expand by node type.
    Semantic zoom the network
    Nodes, line and text size will stay same, only distance between nodes will change.
